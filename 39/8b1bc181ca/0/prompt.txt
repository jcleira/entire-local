# Plan: TUI Dashboard Redesign вҖ” Match Web UI

## Context

After the previous checkpoint list refactor, several issues remain visible in the TUI:
1. Title shows raw prompt ("Implement the following plan:") instead of the git commit message
2. Time displays as "unknown" when `CreatedAt` is zero
3. Missing data compared to web UI (session count, proper stats)
4. UI doesn't fill the full terminal height
5. No visual separation between sections вҖ” needs bordered panels
6. Footer only has keybindings, no summary stats

The goal is to match the entire.io web UI layout while adding bordered "card" panels around each section.

## Changes

### 1. New git reader methods вҖ” `pkg/git/reader.go`

Add three methods to `Reader`:

- **`CommitMessage(hash string) (string, error)`** вҖ” runs `git log -1 --format=%s <hash>`. Returns the commit subject line from the repo (not the orphan branch). Falls back gracefully if hash doesn't exist.
- **`CommitDate(hash string) (time.Time, error)`** вҖ” runs `git log -1 --format=%aI <hash>`, parses RFC3339 output. Used as fallback when metadata `created_at` can't be parsed.
- **`ListEntries(path string) ([]string, error)`** вҖ” runs `git ls-tree --name-only <branch>:<path>`. Returns entry names. Used to count session directories.

### 2. Extend data types вҖ” `pkg/checkpoint/types.go`

**`CheckpointSummary`**: Add fields:
- `CommitMessage string` (after `CommitHash`)
- `SessionCount int` (after `Deletions`)

**`OverviewStats`** (`pkg/checkpoint/stats.go`): Add fields:
- `TotalTokens int`
- `TotalAdditions int`
- `TotalDeletions int`

In `ComputeStats()`, accumulate these totals from the existing loop (totalTokens is already computed but only used for averaging).

### 3. Load new fields вҖ” `pkg/checkpoint/loader.go`

In `loadSummary()`, after existing metadata parsing:
- Call `l.reader.CommitMessage(meta.CommitHash)` вҶ’ set `summary.CommitMessage`
- If `createdAt.IsZero()`, call `l.reader.CommitDate(meta.CommitHash)` as fallback for the timestamp
- Call `l.reader.ListEntries(prefix)`, count entries where `strconv.Atoi(name)` succeeds вҶ’ set `summary.SessionCount`
- All errors handled gracefully (leave defaults)
